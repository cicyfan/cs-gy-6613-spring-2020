    What you need to know on MDP & RL

    Read the web site notes.

    You need to read Section 17.1-17.3 of the AIMA book

    1. You need to be able to determine the transition function of environments like in Fig 17.1.
    2. You need to be able to explain why the policy behaves with the reward function if you are given an environment like in Fig 17.2.
    3. You need to be able to classify an problem if its infinite vs. finite horizon and what is the role of $\gamma$.
    4. You need to know the intuition behind Bellman equations (both expectation and optimality pairs).
      • You need to be able to draw a tree that represents the problem you are trying to solve
    5. You need to be able to explain DP vs MC vs TD.
    6. You need to be able to calculate policy and value iterations.

    Look at exercises like 17.1, 17.2 and 17.8.

    You need to read chapter 21.1-21.5

    1. You need to know how the REINFORCE and SARSA would function in a hypothetical small world.
